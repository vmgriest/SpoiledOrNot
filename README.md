üß† How It Works

SpoiledOrNot uses a Convolutional Neural Network (CNN) with transfer learning to analyze visual features that indicate freshness:

    Fresh foods: Vibrant colors, firm texture, natural appearance

    Spoiled foods: Dullness, mold spots, discoloration, wilting, bruising

The model is trained on curated datasets of fresh and spoiled food images, learning to distinguish subtle visual cues that indicate edibility.
